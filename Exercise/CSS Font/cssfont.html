Here we explore various CSS properties to style font

Font Family

Usage:

Specify multiple fonts to ensure fall back options

p {


font-family:'Arial','Helvetica','sans-serif';


}

Font style

Usage:

For applying different styling to text

p {


font-style: italic;


}

Font Variant

Usage:

To specify the usage of alternate glyphs, such as small-caps, for text.

p {


font-variant: small-caps;


}

Font Weight

Usage:

specifies the weight (or boldness) of the font. It allows you to control how thick or thin the text appears.

p{


font-weight: bold;


}

Font Size

Usage:

To provide specific size

p{


font-size: 30px;


}

Shothand Property

Usage:

For combining styles

p{


font: italic small-caps bold 16px/2 'Georgia', serif;


}
